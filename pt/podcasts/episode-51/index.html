OpenClaw 2026.5.12, Hermes Foundation, Controles de Segundo Plano do Claude Code e Implantações de Agentes Gemini

AgentStack Daily EP051 começa com uma leitura de release do agent-stack: OpenClaw v2026.5.12 reduz as instalações principais, fortalece Telegram, Codex, plugin, gateway, navegador e caminhos de configuração, e melhora a entrega de respostas. Hermes Agent 2026.5.16 adiciona beta nativo para Windows, instalação via PyPI, inicialização mais rápida, um proxy local compatível com OpenAI, upgrades de visão, vídeo, navegador, LSP e verificação. Claude Code 2.1.143 e 2.1.142 apertam dependências de plugins, flags de sessão em segundo plano, comportamento do PowerShell e isolamento de worktree.
