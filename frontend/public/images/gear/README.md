## Image Guidelines
- Use WebP format for best performance
- Target size: 800x600px or 1200x900px
- Keep file sizes under 100KB per image
- Use transparent backgrounds where possible for product shots

## Amazon Image Extraction
For the DualRay pull-up bar specifically:
1. Go to: https://www.amazon.com/dp/B09Z2NHBHJ
2. Right-click on main product image → "Open image in new tab"
3. Download and save as `dualray-pullup-bar.webp` in this folder
4. Or use: Inspector → find `data-old-hires` or high-res image URL
5. Convert to WebP using: https://convertio.co/webp-converter/
