OpenClaw v2026.5.22 और Claude Code 2.1.149 EP057 के अग्रभाग में हैं। OpenClaw गेटवे, प्लगइन, मीटिंग-नोट्स कैप्चर, Discord कॉलबैक, मीडिया/प्रोवाइडर फॉलबैक, सबएजेंट व्यवहार, xAI/Grok सर्च रीयूज़, चैट-सेशन नेविगेशन और पैकेज इंटीग्रिटी को टाइट करता है। Claude Code बेहतर यूसेज अकाउंटिंग, डिफ नेविगेशन, टास्क-लिस्ट रेंडरिंग, क्लाउड MCP कंट्रोल और शेल/सैंडबॉक्स फिक्सेस जोड़ता है। फिर एपिसोड AI न्यूज की ओर बढ़ता है जो स्टैक के लिए मायने रखती है: Gemini मैनेज्ड एजेंट्स, Codex रिमोट वर्क और हाइब्रिड एनवायरनमेंट्स, Anthropic का SDK/MCP अधिग्रहण, AI सिक्योरिटी स्कैनिंग और GitHub-होस्टेड प्रोजेक्ट्स जो एजेंट वर्कफ्लोज़ को बेहतर बना सकते हैं।

[00:00] ओपनिंग: रिलीज़ फिक्सेस, AI न्यूज और रियल प्रोजेक्ट रडार यह एपिसोड OpenClaw v2026.5.22 और Claude Code 2.1.149 से शुरू होता है क्योंकि दोनों रिलीज़ मशीनरी बदलती हैं जिस पर एजेंट स्टैक निर्भर करते हैं: गेटवे स्टार्टअप, प्लगइन मेटाडेटा, मीटिंग नोट्स, Discord कॉलबैक, प्रोवाइडर फॉलबैक व्यवहार, क्लाउड MCP कनेक्टर पॉलिसी, यूसेज अकाउंटिंग, डिफ्स और शेल सेफ्टी। फिर AI-न्यूज लेन Google, OpenAI और Anthropic को कवर करता है क्योंकि हर ने एक ठोस एजेंट इन्फ्रास्ट्रक्चर सरफेस हिलाया: मैनेज्ड रिमोट सैंडबॉक्स, मोबाइल-सुपरवाइज़्ड Codex सेशन, हाइब्रिड/ऑन-प्रेम Codex डिप्लॉयमेंट, SDK/MCP जेनरेशन और फ्रंटियर-मॉडल सिक्योरिटी स्कैनिंग। प्रोजेक्ट लेन Semantic कोड कॉन्टेक्स्ट, करंट डॉक्स, मॉडल रूटिंग, MCP बिल्डिंग, लोकल एजेंट्स, रोल पैक्स और सिक्योरिटी स्कैनर्स के आसपास GitHub-होस्टेड टूल्स पर फोकस्ड रहती है।

[02:00] OpenClaw v2026.5.22 और Claude Code 2.1.149 रिलीज़ रीडआउट OpenClaw का अपडेट व्यापक है, लेकिन उपयोगी थीम एजेंट स्टैक के जहाँ अक्सर रुकावट होती है वहाँ रिलायबिलिटी और कैपेबिलिटी है। गेटवे प्रोसेस-स्टेबल चैनल कैटलॉग रीड्स और प्लगइन मेटाडेटा स्नैपशॉट रीयूज़ पर निर्भर करता है, जिससे स्टार्टअप और स्टेटस वर्क कम बर्वादी होती है। स्टार्टअप-आइडल प्लगइन वर्क सुस्त है, बेमतले Linuxbrew PATH प्रोब्स को स्किप किया जाता है, और कोर गेटवे मेथड हैंडलर्स प्लस पब्लिक-सरफेस एलियास मैप्स गेटवे पाथ को अधिक सुसंगत बनाते हैं। मीटिंग नोट्स को असली अपग्रेड मिलता है: एक्सटर्नल प्लगइन्स और सोर्स प्रोवाइडर्स के पास अब साफ़ कॉन्ट्रैक्ट है, कैप्चर कॉन्फ़िग से ऑटो-स्टार्ट कर सकता है, मैनुअल इम्पोर्ट्स सपोर्टेड हैं, रीड-ओनली CLI एक्सेस मौजूद है, और Discord वॉइस को साइड पाथ की बजाय फर्स्ट लाइव सोर्स के रूप में ट्रीट किया जाता है।

एजेंट और प्लगइन सरफेस भी हिलते हैं। OpenClaw डिफ़ॉल्ट सबएजेंट बूटस्ट्रैप को सबसे मायने रखने वाली फ़ाइलों तक ट्रिम करता है, नेटिव सबएजेंट कम्प्लीशन हैंडऑफ़ फिक्सेस जोड़ता है, प्लगइन SDK में जेनरिक चैनल-मैसेज पोल सेंडिंग और सेशन वर्कफ्लो हेल्पर्स जोड़ता है, और एम्बेडिंग-प्रोवाइडर कैपेबिलिटी कॉन्ट्रैक्ट्स को क्लियर करता है। चैट कंट्रोल सरफेस को सेशन पिकर में सर्च और लोड मोर पैजिनेशन मिलता है, जो तब मायने रखता है जब गेटवे के पास असली बैकलॉग हो। Discord कंपोनेंट कॉलबैक्स के पास अब बाउंडेड TTL है, जो उस जोखिम को कम करता है कि पुराने रिव्यू बटन या स्टेल UI कॉलबैक अपनी उपयोगी लाइफटाइम से आगे लाइव रहें। प्रोवाइडर हैंडलिंग साफ़ होती है: xAI OAuth का Grok web_search के लिए रीयूज़ किया जा सकता है, मॉडल एलियासेस और ऑपरेशन टाइमआउट्स बेहतर होते हैं, Antigravity CLI कॉन्फ़िगर्ड प्रोवाइडर APIs के बाद लोअर-प्रायोरिटी इमेज/वीडियो फॉलबैक बन जाता है, और Codex API-की इमेज जेनरेशन नेटिव OpenAI Images API का उपयोग करता है। डिपेंडेंसी रिफ्रेश, protobufjs 8.4.0, लॉक्ड डिपेंडेंसी वर्क, कैटलॉग प्रूनिंग, सेशन राइट-लॉक क्लीनअप, स्ट्रिक्ट vLLM टूल-फ्री टर्न्स, Telegram टॉपिक फिक्सेस, लोकल Chrome/Ollama प्रॉक्सी बाइपास फिक्सेस जोड़ें, और यह एक असली मेंटेनेंस-और-कैपेबिलिटी रिलीज़ है।

Claude Code का 2.1.149 अपडेट छोटा है, लेकिन यह डेली यूज़ पर असर डालता है। /usage कैटेगरी के हिसाब से लिमिट यूसेज दिखा सकता है, जिसमें स्किल्स, सबएजेंट्स, प्लगइन्स और पर-MCP-सर्वर कॉस्ट शामिल हैं। /diff डिटेल व्यू कीबोर्ड स्क्रॉलिंग सपोर्ट करता है। मार्कडाउन आउटपुट GitHub-फ्लेवर्ड टास्क-लिस्ट चेकबॉक्सेस रेंडर करता है। एंटरप्राइज एडमिन्स को allowAllClaudeAiMcps मैनेज्ड सेटिंग मिलती है जो मैनेज्ड MCP कॉन्फ़िग के साथ claude.ai क्लाउड MCP कनेक्टर्स लोड करने के लिए है। फिक्सेस सिक्योरिटी स्टोरी हैं: PowerShell परमीशन बाइपासेस बिल्ट-इन डायरेक्टरी-चेंजिंग फंक्शन्स के ज़रिए, सैंडबॉक्स राइट अलाउलिस्ट्स जो गलती से बहुत ज़्यादा git वर्कट्री कवर कर लेते थे, PowerShell प्रीफिक्स/वाइल्डकार्ड रूल बग्स, PWD/OLDPWD/DIRSTACK के आसपास स्टेल वेरिएबल ट्रैकिंग, एक macOS find फेलियर जो बड़ी डायरेक्टरीज़ पर फ़ाइल टेबल्स खत्म कर सकता था, और कई रिमोट-सेशन, मैनेज्ड-सेटिंग्स, टेलीमेट्री-हेल्पर, ट्रांसक्रिप्ट और UI फिक्सेस। वर्शन 2.1.150 इंटरनल इन्फ्रास्ट्रक्चर ओनली है, इसलिए यूज़र-फेसिंग डेल्टा 2.1.149 है भले ही इंस्टॉल्ड पैकेज नया हो।

[10:00] Gemini 3.5 Flash और Gemini API मैनेज्ड एजेंट्स Google का Gemini 3.5 Flash एनाउंसमेंट एजेंट इन्फ्रास्ट्रक्चर न्यूज के रूप में देखने लायक है, सिर्फ मॉडल मार्केटिंग नहीं। Google कहता है 3.5 Flash एजेंटिक वर्कफ्लोज़ और कोडिंग के लिए बनाया गया है, जिसमें Terminal-Bench, GDPval-AA, MCP Atlas, मल्टीमॉडल, स्पीड और लॉन्ग-होराइज़न-टास्क क्लेम्स मज़बूत हैं। ज़रूरी सवाल यह नहीं है कि कौन सा बेंचमार्क हफ्ते जीतता है। सवाल यह है कि क्या एक फास्ट मॉडल लंबे, टूल-हेवी लूप्स में पर्याप्त अच्छा रह सकता है जहाँ लैटेंसी और सुपरविज़न मायने रखते हैं।

Gemini API मैनेज्ड एजेंट्स लॉन्च इसे ठोस बनाता है। एक सिंगल कॉल Antigravity-पावर्ड एजेंट को आइसोलेटेड, एफ़ेमरल Linux एनवायरनमेंट में स्पिन अप कर सकता है। एजेंट रीज़न कर सकता है, टूल्स कॉल कर सकता है, कोड एक्ज़ीक्यूट कर सकता है, फ़ाइल्स मैनेज कर सकता है और वेब ब्राउज़ कर सकता है। फॉलो-अप कॉल्स एनवायरनमेंट रीयूज़ कर सकते हैं, इसलिए स्टेट जारी रह सकता है बजाय इसके कि हर रिक्वेस्ट एक फ्रेश स्टेटलेस प्रॉम्प्ट बने। डेवलपर्स AGENTS.md और SKILL.md-स्टाइल फ़ाइल्स में इंस्ट्रक्शन्स, स्किल्स और डेटा के साथ कस्टम एजेंट्स डिफ़ाइन कर सकते हैं। यह सीधे लोकल OpenClaw और Hermes सोच के लिए रिलेवेंट है: क्या लोकल रहना चाहिए क्योंकि यह क्रेडेंशियल्स या प्राइवेट कोड को टच करता है, और क्या सुरक्षित रूप से मैनेज्ड सैंडबॉक्स में मूव किया जा सकता है क्योंकि एनवायरनमेंट कंट्रोल्ड, डिस्पोज़ेबल और स्केल करना आसान है?

[17:00] Codex रिमोट सुपरविज़न, एक्सेस टोकन्स और हाइब्रिड एनवायरनमेंट्स OpenAI का Codex डायरेक्शन सुपरविज़न और डिप्लॉयमेंट शेप के बारे में है। ChatGPT मोबाइल ऐप में Codex किसी यूज़र को Mac या रिमोट एनवायरनमेंट पर चल रहे एक्टिव वर्क से कनेक्ट करने, लाइव प्रोजेक्ट स्टेट देने, कमांड्स अप्रूव करने, टर्मिनल आउटपुट, स्क्रीनशॉट्स, टेस्ट रिज़ल्ट्स और डिफ्स रिव्यू करने, और होस्ट मशीन पर न होकर टास्क रीडायरेक्ट करने देता है। यह मायने रखता है क्योंकि लॉन्ग-रनिंग कोडिंग एजेंट्स सिर्फ इंटेलिजेंस की कमी से नहीं फेल होते। वे फेल होते हैं क्योंकि वे डिसीज़न पॉइंट पर पहुँचते हैं और इंतज़ार करते हैं, या गलत फोर्क ले लेते हैं क्योंकि कोई इंसान वहाँ नहीं था जो थोड़ा सा जजमेंट दे।

एंटरप्राइज़ साइड उतना ही ज़रूरी है। रिमोट SSH जनरली अवेलेबल है, Codex मैनेज्ड रिमोट एनवायरनमेंट्स के अंदर काम कर सकता है, हुक्स प्रॉम्प्ट्स को सीक्रेट्स के लिए स्कैन कर सकते हैं या वैलिडेटर्स रन कर सकते हैं, और प्रोग्रामैटिक एक्सेस टोकन्स नॉन-इंटरैक्टिव वर्कफ्लोज़ को स्कोप्ड वर्कस्पेस आइडेंटिटी देते हैं। Dell पार्टनरशिप हाइब्रिड और ऑन-प्रेम एनवायरनमेंट्स की ओर इशारा करती है जहाँ कोड, डेटा, पॉलिसीज़ और अप्रूव्ड कंप्यूट पहले से मौजूद हैं। यह Codex की असली कहानी है: एक कोडिंग एजेंट जो कहीं से भी सुपरवाइज़ किया जा सकता है जबकि फ़ाइलों, क्रेडेंशियल्स और कंट्रोल्स के पास चल रहा है जिनकी उसे ज़रूरत है।

[24:00] Anthropic स्टेनलेस और प्रोजेक्ट ग्लासविंग Anthropic का स्टेनलेस अधिग्रहण एक एजेंट-कनेक्टिविटी स्टोरी है। स्टेनलेस API स्पेसिफिकेशन्स को SDKs, CLIs और MCP सर्वर्स में बदलता है जो कई भाषाओं में फैलते हैं, और Anthropic कहता है कि स्टेनलेस Claude API के शुरुआत से ही ऑफिशियल Anthropic SDKs जेनरेट करता रहा है। एजेंट्स के लिए यह मायने रखता है क्योंकि एक एजेंट उतना ही उपयोगी है जितना वह सिस्टम्स को सुरक्षित और सही तरीके से पहुँच सकता है। क्लीन SDKs, CLIs और MCP सर्वर्स बोरिंग रैपर वर्क नहीं हैं; ये हैंडल्स हैं जिनका उपयोग एजेंट्स कार्य करने के लिए करते हैं।
